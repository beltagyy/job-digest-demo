# Job Digest Demo Website

Frontend showcase website for [Job Digest](https://github.com/beltagyy/job-digest) - an AI-powered job search assistant.

Built with **Astro** + **Tailwind CSS** and deployed to GitHub Pages.

## 🚀 Quick Start

```bash
git clone https://github.com/beltagyy/job-digest-demo.git
cd job-digest-demo
npm install
npm run dev
```

## 📁 Project Structure

```
src/
├── layouts/Layout.astro       # Main layout
├── pages/
│   ├── index.astro           # Landing page
│   ├── digest.astro          # Sample digest
│   └── how-it-works.astro    # How it works guide
└── components/
    └── WaitlistForm.astro    # Signup form
```

## 📝 Learn Astro

- **File-based routing**: Files in `src/pages/` automatically become routes
- **Components**: Create `.astro` files with HTML + CSS + minimal JS
- **Layouts**: Wrap pages with `<slot />`
- **Tailwind**: Already integrated, use classes directly
